Padronização 

1 - Estados, nas rotas(url) sempre usar siglas em maiúsculo, exemplo: "/CE/", 
usar a função "converte_sigla_em_nome" para converter no nome do estado

2 - Datas, sempre nas rotas usar o formato "jan-2019", usar as funções 
"pega_mes" para converter a string "jan-2019" em "janeiro" (Para base de estados)
"pega_ano" para converter a string "jan-2019" em "2019" (Para base de estados)
"converte_para_data" para converter a string "jan-2019" em "2019-01-01" (Para base de cidades)

3 - Para crimes e cidades, com espaço separar com "-", exemplo: Homicídio doloso -> Homicídio-doloso
